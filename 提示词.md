# FQinCatcher

帮我编写一个适用于Minecraft Fabric 1.21.1的Mod，用于快速复制MC服务器的地图。

Mod名称：FQinCatcher
主类名：cn.craftimr.fQinCatcher.FQinCatcher
适用游戏版本：Java版1.21.1，Fabric加载器
Java版本：Java21
构建器：Gradle

可供查阅的代码位于`./.roo/lib/`
    `./.roo/lib/minecraft/1.21.1/`：原版Minecraft 1.21.1的客户端与服务端源码
    `./.roo/lib/SeedcrackerX`：一个Fabric端的用于破解服务器地图种子的工具，其中-master是仓库的最新版本，-2.15.0是适用于Minecraft Fabric 1.21.1的版本。

使用逻辑：
第一步：安装该Mod并加入多人游戏服务器
第二步：按指定的按键打开该Mod的GUI页面
GUI页面内容：
**服务器复制列表**
    注：点击列表中的地图以选中该地图
    **侧边栏**
        新建地图
        删除（选中的）地图
    **底部栏**（针对已选中地图）
        **地图种子相关**
            服务器种子破解（按键）
            服务器种子（文本框）
            保存（保存种子到选中的地图，按键）
        **复制相关**
            生成地图（根据种子生成镜像地图，按键）（镜像地图指的是模拟玩家正在地图的这个位置加载区块以生成）
            开始复制（开始复制该服务器地图到选中的地图，按键）
            暂停复制（按键）

服务器种子破解：
    点击后自动根据SeedcrackerX的逻辑破解种子，将结果显示在聊天框并自动保存。
复制相关：
    将服务端发送到客户端的地图数据复制到选中的地图，包括方块、实体、生物群系等。
